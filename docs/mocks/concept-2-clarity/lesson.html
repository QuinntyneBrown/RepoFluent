C#/.NET 
public sealed class CartPricingService : ICartPricingService
{
    private readonly IPromotionRuleRegistry _rules;
    private readonly ITaxCalculator _tax;

    public CartPricingService(IPromotionRuleRegistry rules, ITaxCalculator tax)
    {
        _rules = rules;
        _tax = tax;
    }

    public async Task<CartTotals> PriceAsync(Cart cart, CancellationToken ct)
    {
        var subtotal = cart.Lines.Sum(l => l.UnitPrice * l.Quantity);
        var discounted = await _rules.ApplyAsync(cart, subtotal, ct);
        var tax = _tax.Calculate(discounted, cart.ShippingAddress);
        return new CartTotals(subtotal, discounted, tax);
    }
}

Glossary

Pricing rule

A pricing rule is a small, ordered policy that adjusts a cart subtotal before tax. In Orion, rules are registered centrally so Checkout can explain which promotion changed the total.
