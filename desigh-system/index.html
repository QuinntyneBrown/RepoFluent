RepoFluent / Code Command / v0.1

Learn the system.
Trust the interface.

A dense, keyboard-first product language for turning unfamiliar enterprise codebases into navigable learning environments. This is the visual and interaction contract for every RepoFluent surface.

Principles

01 / CONTEXT

Keep learning beside the code

Split views, inspectors, breadcrumbs, and source anchors preserve the learner's place while they investigate.

02 / EVIDENCE

Make progress explainable

Completion, mastery, scores, recency, and provenance remain distinct and expose the evidence behind them.

03 / CONTROL

Reward keyboard fluency

Commands are discoverable, focus is visible, actions are reversible, and no core capability depends on a pointer.

04 / DENSITY

Compact without becoming cryptic

Technical teams get information-rich layouts, consistent metrics, plain labels, and generous reading line-height.

05 / SAFETY

Show boundaries and consequences

Permissions, versions, protected answers, destructive actions, and stale source references are explicit.

06 / ACCESS

Equivalent by default

Maps, charts, motion, and color always have semantic, keyboard-operable, text-based equivalents.

Implementation contract

TOKENS.CSS authoritative

Semantic --rf-* values for color, type, spacing, geometry, elevation, motion, charts, and layers.

COMPONENTS.CSS product

Framework-neutral .rf-* classes whose states and behaviors are demonstrated throughout these docs.

DOCS.* documentation only

.ds-* chrome, navigation, search, and example framing are explicitly outside the product contract.

Standalone and durable

All pages work over file://, use bundled SVG icons, rely on system fonts, and require no build step.
