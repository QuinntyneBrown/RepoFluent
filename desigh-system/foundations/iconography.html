Accessible semantics

DECORATIVE VS. INFORMATIVE
Source revision is stale
  • Hide decorative icons from assistive technology when adjacent text supplies the name.
  • Icon-only buttons require an explicit accessible name and a visible tooltip on hover and focus.
  • Informative status icons receive a label and are paired with visible status text.

Usage rules

DO

Use one stable icon for one action throughout the product and pair unfamiliar actions with text.

DON'T

Mix emoji, platform glyphs, filled illustrations, or different icons for the same command.
