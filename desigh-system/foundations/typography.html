# ORDER PROCESSING FOUNDATIONS

How an order becomes a workflow

The API accepts the request quickly. The real work begins after it publishes a message. This wider line-height supports sustained reading while the compact surrounding shell preserves technical context.

  • Keep lesson prose between 55 and 80 characters per line.
  • Use sentence case for headings and actions; reserve uppercase for short overlines and state codes.
  • Never place long explanatory paragraphs in the mono family.
