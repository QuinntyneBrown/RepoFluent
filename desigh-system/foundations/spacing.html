Region padding

DENSE / 8–12
Trees, toolbars, metadata, status, and code gutters.
STANDARD / 16–24
Cards, dialogs, forms, inspectors, and dashboard modules.
READING / 32–64
Lesson documents, page introductions, and section separation.

Responsive grid

ClassViewportBehavior
Wide workstation≥ 1024pxTitlebar + activity bar + sidebar + multi-pane workbench
Tablet / compact768–1023pxSidebar may collapse; split panes retain usable minimum widths
Phone< 768pxSingle content column, drawer navigation, stacked inspectors
Narrow phone< 480pxIcon-only global search, overflow-safe tables and code

Responsive behavior rearranges information; it does not remove required explanations, assessment rules, or states.

Density rules

DO

Use compact rows for scanning and larger padding for comprehension-heavy content.

DON'T

Shrink essential pointer targets below 36px unless the surrounding grouped target remains at least 44px.
