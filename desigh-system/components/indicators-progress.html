Component / C09

Indicators & progress

State indicators answer “what condition is this in?” Progress answers “how much of a defined journey is complete?” Mastery is evidence, not generic progress.

Usage rules

DO

Distinguish completion, score, mastery, recency, and required/optional status.

DON'T

Collapse all learning evidence into one unexplained number or animate progress without a real update.
