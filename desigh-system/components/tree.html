Keyboard and focus

  • ↑/↓ move among visible nodes; → expands or enters a branch; ← collapses or moves to its parent.
  • Home/End move to first/last visible node; typeahead moves to a matching label.
  • Enter opens the selected content. Space selects when selection is separate from opening.
  • Virtualized trees retain the active descendant and announce result count after filtering.

Search and large trees

FILTER PRESERVES ANCESTRY

Filtering keeps the ancestor path visible, emphasizes matched text without changing accessible names, and provides a clear reset action.

Usage rules

DO

Use a tree for true parent-child hierarchy where expansion preserves meaningful context.

DON'T

Use a tree as a visual list, nest more than needed, or make the disclosure icon the only usable target.
