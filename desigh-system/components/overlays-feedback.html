Tooltips

SUPPLEMENTAL LABEL
Copy source link

Tooltips appear on hover and focus after a short delay, remain under pointer movement, and contain only a concise label or shortcut—not required instructions.

Overlay rules

  • Allow only one modal layer. A menu or tooltip closes before its parent dialog closes.
  • Trap focus within a modal dialog; inert the background; close with Escape unless work would be lost.
  • Use a full-page state for complex multi-step work instead of a very large dialog.
  • Associate field-level errors with fields and reserve banners for page- or workflow-level conditions.
